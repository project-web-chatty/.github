## PROJECT INTRODUCTION

Web CHATTY 프로젝트는 웹소켓 및 메시지 브로커 기술 스택을 활용하여 슬랙과 디스코드 등의 웹 메신저 기능을 모방한 웹 서비스를 만들고자 시작한 프로젝트 입니다. 워크 스페이스 생성, 채널 생성, 채널 내 그룹 채팅, 프로필 관리 등의 서비스 구현을 본 프로젝트의 주 목표로 하였습니다.
